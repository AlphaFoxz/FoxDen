# FoxDen App Admin

Admin application module for FoxDen - the main Spring Boot application entry point.

## Package
`com.github.alphafoxz.foxden.app.admin`

## Contents
- `FoxdenAdminApplication.kt` (main entry point)
- All controllers
- All configurations (JimmerConfig, SpringSecurityConfig, etc.)
- `application.yaml` and `application-dev.yaml`
- Static resources

## Running the Application

```bash
./gradlew :foxden-app:foxden-app-admin:bootRun
```

## Building

```bash
./gradlew :foxden-app:foxden-app-admin:build
```
